# Calculadora de Desconto por Mesa

## Sobre o Projeto

Este programa em C calcula o valor final a ser pago por uma mesa em um restaurante, aplicando um desconto baseado no n√∫mero da mesa.

**Regras de Desconto:**
* **Mesas 1 a 3:** Desconto de 15% (0.15).
* **Mesas 4 a 7:** Desconto de 12.5% (0.125).
* **Outras Mesas:** Sem desconto (0%).
